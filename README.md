# Credit Card Fraud Detection

## Overview

This project analyzes large and highly imbalanced credit card transaction data to identify fraudulent transactions utilizing various machine learning classification techniques.

The goal is to build predictive models capable of distinguishing fraudulent activity from legitimate transactions while evaluating performance using appropriate classification metrics.

## Dataset

- Credit Card Fraud Detection Dataset
- Source: Kaggle
- Transactions labeled as fraudulent or legitimate

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Models

- Logistic Regression
- Random Forest Classifier

## Data Preprocessing

- Missing value analysis
- Feature scaling
- Train-test split
- Handling class imbalance

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Key Findings

The models were evaluated on their ability to identify rare fraudulent transactions while minimizing false positives.
